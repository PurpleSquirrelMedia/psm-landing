Frequently Asked Questions

How do I become a creator on Purple Squirrel Media?

Simply sign up for an account and you can immediately start uploading content. Our platform supports HLS streaming with automatic transcoding and AI-generated captions.

What video formats do you support?

We accept MP4, MOV, AVI, MKV, and WebM formats. All videos are automatically transcoded to HLS for adaptive streaming across all devices.

How does the NFT membership work?

Our Solana-based NFT memberships give you permanent access to premium features. No recurring payments - own your membership forever and even resell it if you want.

Is there an API for developers?

Yes! Our REST API allows you to upload videos, manage transcoding, retrieve captions, and more. Check out our documentation for full API reference.

What cloud providers do you use?

We use a multi-cloud architecture: Oracle Cloud for compute and storage, Azure for web hosting and CDN, IBM Watson for AI captions, and Google Cloud for analytics.
